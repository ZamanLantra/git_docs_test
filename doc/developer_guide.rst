Basic Code Block
----------------

You can include a basic code block like this:

::

    # This is a basic code block
    print("Hello from a basic code block!")

-----------------------------------

.. code-block:: python

    def add(a, b):
        return a + b

    result = add(2, 3)
    print(result)
